WindowsGSM.Windrose

WindowsGSM plugin for Windrose Dedicated Server.

ENGLISH

WindowsGSM Installation
1. Download WindowsGSM from https://windowsgsm.com/
2. Create a folder where you want WindowsGSM and your servers to live.
3. Place WindowsGSM.exe inside that folder and run it.

Plugin Installation
1. Download this repository or the latest release.
2. Extract it.
3. Move the Windrose.cs folder into the WindowsGSM plugins folder.
4. You can also import the ZIP from the puzzle icon inside WindowsGSM.
5. Reload plugins or restart WindowsGSM.
6. Go to Servers, click Install Game Server, and select Windrose Dedicated Server.

Server Installation
1. Create a new Windrose server in WindowsGSM.
2. Click Install to download the dedicated server with SteamCMD.
3. When the installation finishes, start the server once.
4. After the first successful start, stop the server from WindowsGSM.
5. This step is required so Windrose creates its server files and the data appears in the Parameters section.
6. After that, you can edit the available values directly from the panel.

Useful Notes
- Default App ID: 4129620
- Main config: R5\ServerDescription.json
- World data is created after the first successful start.

ESPAÑOL

Instalacion de WindowsGSM
1. Descarga WindowsGSM desde https://windowsgsm.com/
2. Crea una carpeta donde quieras tener WindowsGSM y tus servidores.
3. Coloca WindowsGSM.exe dentro de esa carpeta y ejecutalo.

Instalacion del plugin
1. Descarga este repositorio o el ultimo release.
2. Extraelo.
3. Mueve la carpeta Windrose.cs a la carpeta plugins de WindowsGSM.
4. Tambien puedes importar el ZIP desde el icono de puzzle dentro de WindowsGSM.
5. Recarga los plugins o reinicia WindowsGSM.
6. Ve a Servers, pulsa Install Game Server y selecciona Windrose Dedicated Server.

Instalacion del servidor
1. Crea un servidor nuevo de Windrose en WindowsGSM.
2. Pulsa Install para descargar el servidor dedicado con SteamCMD.
3. Cuando termine la instalacion, inicia el servidor una vez.
4. Despues del primer arranque correcto, deten el servidor desde WindowsGSM.
5. Este paso es obligatorio para que Windrose genere sus archivos y los datos aparezcan en la seccion Parameters.
6. A partir de ahi, podras modificar los valores disponibles directamente desde el panel.

Notas utiles
- App ID por defecto: 4129620
- Config principal: R5\ServerDescription.json
- Los datos del mundo se generan despues del primer arranque correcto.
